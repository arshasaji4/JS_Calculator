# Calculator App

A simple and stylish Calculator App built using HTML, Tailwind CSS, and JavaScript.
This calculator can perform basic arithmetic operations like addition, subtraction, multiplication, division, and percentage calculations.

## Features

* Clean and responsive UI
* Addition, subtraction, multiplication, division
* Percentage calculation
* Delete last character
* Clear all input
* Error handling using `try...catch`
* Glassmorphism design using Tailwind CSS

## Technologies Used

* HTML5
* Tailwind CSS
* JavaScript
